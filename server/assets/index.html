Vercel AI SEO Optimization Agent

An agent built with Nitro, Vercel AI SDK, and Vercel AI Gateway that reviews your content and researches related web insights to improve novelty and stay competitive in AI-driven search.

API Endpoint POST /api/analyze
Example Request 
curl -X POST "http://localhost:3000/api/analyze?agent-secret=your-secret" \
  -H "Content-Type: application/json" \
  -d '{"content": "Your content to analyze..."}'
